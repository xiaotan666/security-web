# security-web

#### 介绍
一个基于java web层的安全权限认证框架

#### 软件架构
基于JDK17开发
目前已适配springboot3.0+
支持分布式微服务系统
支持基于redis的数据缓存，默认数据缓存至内存中

#### 目前支持功能
1.  登录认证（支持自定义、多种认证方式）
2. 基于登录鉴权
3. 角色鉴权
4. 权限码鉴权
5. 支持自定义鉴权方式

#### 待完善
1.  oauth2的支持，目前还在开发中（暂不能使用）
2. 更多web框架适配
3. bug待测试


#### 安装教程

1.  拉取分支代码
2.  直接通过maven引入项目或者编译为jar包在引入到项目中即可

#### 使用说明

1.  引入依赖
2.  使用@EnabledSecurity注解标注至主程序入口中
3.  启动项目即可
